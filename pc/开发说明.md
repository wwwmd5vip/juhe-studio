## 🚀 快速开始

### 环境要求
- Node.js ≥ 20
- pnpm ≥ 10.27.0（项目指定包管理器）

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
# 启动开发服务器（含 API 生成）
pnpm dev

# 或带热重载
pnpm dev:watch
```

### 构建
```bash
# 全平台构建
pnpm build

# 指定平台
pnpm build:mac    # macOS
pnpm build:win    # Windows
pnpm build:linux  # Linux
```

### 测试
```bash
# 运行全部测试
pnpm test

# 指定模块测试
pnpm test:main
pnpm test:renderer
pnpm test:aicore
```

### 代码检查
```bash
pnpm lint          # 自动修复
pnpm format        # 格式化
pnpm typecheck     # 类型检查
pnpm i18n:check    # 国际化检查
```

## 🎯 核心功能速览

### 图像创作
- 文生图 / 图生图（支持多参考图组合）
- 智能修图、局部重绘、智能扩图
- 一键抠图、高清放大、生成变体

### 视频创作
- 文生视频 / 图生视频 / 视频+图文生视频
- 首尾帧生视频
- 多参考图组合生视频

### 工作流
- 无限画布节点编排
- 节点间自动连线与数据传递
- 工作流保存/复用/分享

### 辅助工具
- 提示词自动优化与拓写
- 预设风格/镜头/光影/画质标签
- 提示词模板库
- Prompt 反推（图/视频 → 提示词）
- 创作助手（电商助手等）

### 文本功能
- AI 聊天窗口（支持 Markdown/代码高亮）
- 自定义智能体
- 深度研究（多轮研究生成报告）

### 电商专用
- AI 穿衣/试衣（多角度模特）
- 电商套图生成（1张产品图 → 4张场景图）

## 📝 开发规范

### 代码提交
- 遵循 Conventional Commits 规范
- 提交前运行 `pnpm lint` 和 `pnpm test`

### 国际化
- 所有用户可见字符串必须走 i18n，禁止硬编码
- 新增字符串同步到 `src/renderer/src/i18n/locales/`
- 使用 `pnpm i18n:sync` 同步语言文件

### 样式规范
- 新组件优先使用 Tailwind CSS + Shadcn UI
- 遗留 Ant Design / styled-components 代码逐步迁移
- 使用 `pnpm styles:canonical` 修复 Tailwind 类名规范
