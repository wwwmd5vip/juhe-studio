# 核心任务：自动识别产品外观并生成标准化尺寸标注SKU图
## 1. 产品识别与形态还原
- 自动分析输入图片中的产品：识别包装形态（袋装/盒装/瓶装等）、主体结构、视觉比例
- 保留产品原有设计细节（文字、图案、材质质感），仅调整立体角度为45°斜视角，确保产品完整展示在画面中心
- 若用户填写了上方自定义尺寸，则**完全覆盖**原图尺寸，按新数据重建产品比例；若未填写，则提取原图标注的尺寸数据

## 2. 图像基础规范
- 背景：纯白色（RGB: 255,255,255），无阴影、无杂边、无水印
- 分辨率：300dpi，图片尺寸固定为800×800像素
- 格式：PNG（透明通道可选，默认不透明）
- 产品占比：产品主体占画面60%-70%，四周留白均匀

## 3. 尺寸标注规范（强制执行）
- 字体：黑色、12号、宋体（SimSun），字重常规
- 位置：每条尺寸边旁**距离产品边缘10像素**，文字方向与边平行，不遮挡产品主体
- 对应关系：
  - 长：水平方向前后边
  - 宽：水平方向左右边
  - 高：垂直方向上下边
- 标注格式：「边名称: 数值cm」（例如：长: 20cm），文字清晰易读

## 4. 执行流程
1.  解析输入：识别产品外观 + 提取/接收上方自定义尺寸数据
2.  建模：按尺寸比例重建产品立体模型，保留包装细节
3.  构图：将产品放置在800×800白色背景正中心
4.  标注：按规范在对应边旁添加尺寸文字
5.  输出：导出为300dpi、800×800像素的PNG格式图片
