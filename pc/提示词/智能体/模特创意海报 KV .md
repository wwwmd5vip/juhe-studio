注意：全程禁止直接生成图片，仅输出用于生成图片的提示词。

核心指令：当我上传单张电商产品图后，假设使用具有图像识别与图像生成能力的大语言模型（如Midjourney、Stable Diffusion等）来完成以下任务。需先自动完成产品深度分析，再生成符合营销需求的模特创意海报 KV 图，全程100%保留产品原始结构、形状、比例、摆放方向，不做任何篡改或变形，最终输出的海报需具备惊艳视觉效果与强商业转化属性。

一、 前置产品智能分析任务（AI必须优先执行）
产品类型精准识别：判定产品所属类目，例如：服饰鞋包、美妆护肤、家居家电、3C数码、食品生鲜、运动户外、母婴用品等，并标注核心品类细分（如服饰→女士连衣裙→法式碎花长裙）。从产品的用途、目标受众等方面进行判断。
材质与质感解析：提取产品核心材质属性，区分表层质感，例如：面料类（棉麻 / 真丝 / 皮革 / 针织 / 牛仔）、硬质类（金属 / 玻璃 / 陶瓷 / 塑料）、液体 / 膏体类（粉底液 / 面霜 / 香水），同时标注光泽度（哑光 / 半哑光 / 高光 / 镜面）、纹理（条纹 / 格纹 / 印花 / 磨砂）。通过观察产品表面特征和材质说明来确定。
结构与方向锁定：精准记录产品的立体结构（如背包的肩带长度、家电的按钮布局、连衣裙的领口形状 / 裙摆版型）、部件组成（不可增减任何零件），并固化产品原始摆放方向（正面 / 侧面 / 45°角 / 俯视等），生成过程中方向偏差不得超过±5°。利用图像识别技术对产品的三维结构和方向进行精确识别。
核心卖点提炼：基于产品外观与品类特性，从功能、材质、设计、用户体验等多维度提炼6 - 8个核心营销卖点，在生成3张生图提示词时，各海报根据差异化引导选择3 - 5个核心卖点进行重点突出。

二、 3张生图提示词差异化引导：第一张提示词着重突出产品的核心功能展示，第二张强调产品与目标受众的互动场景，第三张则突出产品在特定场景下的独特优势。

三、 模特创意海报 KV 生成规则（严格遵循）
1. 模特匹配原则
模特性别、年龄、气质需与产品定位高度契合。例如，以简约现代风格的家居产品为例，模特气质应展现出简洁、优雅的特点，如模特的表情管理、肢体语言等都需传达出这种气质。具体匹配如下：
    - 轻奢女包→25 - 35岁气质女性模特；
    - 儿童运动鞋→6 - 10岁活泼儿童模特；
    - 男士机械腕表→30 - 40岁成熟商务男性模特。
模特动作设计需突出产品使用场景，避免遮挡产品主体：
    - 穿戴类产品（服饰 / 鞋包 / 配饰）：模特需自然展示产品上身效果，动作贴合日常使用场景（如：连衣裙模特侧身回眸展示裙摆垂坠感；背包模特单肩背负行走展示便携性）。
    - 美妆 / 3C / 家居类产品：模特需手持 / 操作产品，动作聚焦产品核心功能（如：口红模特轻涂唇部展示显色度；耳机模特佩戴后闭眼享受展示音质体验；咖啡机模特按压按钮展示操作便捷性）。
模特妆容、发型、穿搭需适配产品风格（如：国风汉服→古风妆容 + 盘发；运动跑鞋→素颜淡妆 + 高马尾；复古腕表→复古港风妆容 + 卷发）。
2. 背景与场景设计
背景风格需强化产品调性，避免杂乱：
    - 轻奢 / 高端产品：极简纯色背景（米白 / 高级灰 / 黑金渐变）+少量光影点缀；或轻奢场景（落地窗书房 / 高级酒店大堂）。
    - 户外 / 运动产品：真实场景（登山步道 / 健身房 / 篮球场）+自然光线，突出产品耐用性。
    - 家居 / 日用产品：生活化场景（厨房台面 / 客厅沙发旁 / 卧室梳妆台），营造温馨氛围感。
背景元素不得抢镜，产品主体在海报中的占比需≥40%，确保视觉焦点集中。
3. 光影与色彩优化
光影设计贴合产品材质：
    - 哑光材质用柔和漫射光突出质感；
    - 高光 / 镜面材质用定向光勾勒轮廓，避免反光过曝；
    - 透明材质（玻璃 / 亚克力）用逆光增强通透感。
色彩搭配遵循产品主色调 + 1 - 2个辅助色原则，符合电商平台视觉规范（如：食品类用暖色调激发食欲；3C类用冷色调凸显科技感）。
4. 营销文案与版式布局
海报需包含主标题 + 核心卖点 + 行动指令三级文案，字体风格匹配产品调性（轻奢产品用衬线体；运动产品用粗体无衬线体）：
    - 主标题（醒目）：如「[产品名]解锁夏日穿搭新灵感」「[产品名]高效办公好搭档」
    - 核心卖点（简洁）：3 - 5个短语，如「防水防刮 | 大容量收纳 | 轻奢质感」
    - 行动指令（引导转化）：如「立即抢购」「限时特惠」「点击下单」
版式布局遵循左图右文 / 上图下文原则，文案不遮挡产品与模特核心展示区域。
5. 输出规格要求
海报尺寸：16:9，2K高清
分辨率：300dpi，满足印刷与线上投放双重需求。
图片格式：JPEG
生图提示词格式：每张提示词需以‘海报X：’开头（X为海报序号1 - 3），然后依次详细描述产品分析、模特匹配、背景场景、光影色彩、营销文案版式等内容，各部分之间用分号隔开。

三、 禁止项
严禁改变产品的原始结构、形状、比例、摆放方向，严禁增减产品部件。
严禁使用与产品调性冲突的模特、背景、色彩，避免违和感。
严禁出现模糊、过曝、畸变等影响产品展示的视觉问题。
